<p>LoveLaceAV — это сканер вредоносных программ с открытым исходным кодом для Android. Он использует базы сигнатур в стиле ClamAV и является форком (прекратившего существование) приложения Hypatia, созданного Divested Computing Group.</p>

<b>Возможности</b>
<ul>
<li>Почти нулевая трата батареи: вы никогда не заметите никакого влияния на аккумулятор вообще</li>
<li>Очень быстро: он может сканировать небольшие файлы (1 МБ) за 20 мс и даже большие файлы (40 МБ) за 1000 мс.</li>
<li>Эффективен по памяти: с включенными антивирусными базами по умолчанию использует менее 120 МБ.</li>
<li>Регулярное сканирование: разрешение выбора /системы, внутреннего хранилища, внешнего хранилища и установленных приложений</li>
<li>Сканер в реальном времени: может обнаружить вредоносное ПО в режиме реального времени при записи/переименовании во внутреннем хранилище</li>
<li>Полностью автономно: Интернет используется только для загрузки баз данных сигнатур, файлы никогда не покинут ваше устройство</li>
<li>Постоянство: автоматически перезагрузит загрузку/обновление</li>
<li>Маленькая кодовая база: до 1000 строк кода, его можно проверить даже кем-то с базовым опытом программирования</li>
<li>Минимальные зависимости: приложение использует библиотеки только при необходимости</li>
<li>Базы данных сигнатур могут быть включены/отключены по желанию пользователей</li>
</ul>
